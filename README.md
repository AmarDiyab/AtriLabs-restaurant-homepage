# atrilabs_restaurant_homepage

Frontend of a restaurant homepage and deployment using Atri framework

## Deployed app


## How to use

### Step 1:

```shell
git clone https://github.com/AmarDiyab/AtriLabs-restaurant-homepage.git

cd AtriLabs-restaurant-homepage

pipenv install atri
```

### Step 2:

```shell
pipenv shell

atri start
```
